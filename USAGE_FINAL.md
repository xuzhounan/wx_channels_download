## ⚙️ 进阶配置

### 完整参数示例
```bash
sudo ./wx_video_download_final \
  --auto \
  --download-dir="/path/to/your/download/folder" \
  --port=2023 \
  --dev=Wi-Fi
```

### 参数说明
- `--auto`: 启用自动下载模式（必需）
- `--download-dir`: 指定下载目录
- `--port`: 代理端口（默认：2023）
- `--dev`: 网络设备（可选）
